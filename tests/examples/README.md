Sample TEI/XML files for manual testing and future fixture-based tests.

Drop example files in this folder when you want to try parser and segmentation behavior against real inputs.

Current app support:
- chapter-based segmentation
- fixed word-count segmentation

Not implemented yet:
- true page-based segmentation

Suggested filenames:
- `sample_chapters.xml`
- `sample_long_text.xml`
- `sample_edge_cases.xml`

If you add examples here, we can later wire them into automated tests.
